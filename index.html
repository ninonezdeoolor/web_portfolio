Tools

HTML

HTML stands for Hypertext Markup Language and is the foundation of web pages on the Internet. HTML is made up of tags, elements and attributes which define how a webpage is structured, styled and linked. HTML is used by web browsers and web servers to create and display webpages.

CSS

CSS stands for Cascading Style Sheets. CSS is a scripting language used to style HTML elements on a web page. It can be used to control font, color, images, positioning, sizing, and spacing of web page content. CSS is used to create a visually appealing, uniform web page layout that can be quickly loaded and accessed by users.

BOOTSTRAP

Bootstrap is a free and open-source CSS framework for developing responsive, mobile-first, and cross-browser websites. It provides basic CSS styling and components, and a powerful grid system for quickly creating layouts. Bootstrap also includes optional JavaScript plugins for transitions, modal boxes, and more.

NODE.JS

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It is used to create scalable network applications and manipulate, create, and convert data with JavaScript. It provides an event-driven, non-blocking I/O model that makes building fast and efficient web applications easier.
